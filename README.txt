== Description ==

Complete your customers order by using Smart Shipping Tracking Plugin by adding Tracking Numbers of your customers orders.

Using Smart Shipment Tracking you can select shipment company and add tracking number, this will create a notification to the cusotmer on Email and SMS notification.

Upload Bulk Shipment Tracking Numbers by using .csv

Custom email notification related to Tracking company with Direct tracking URL. 

== Installation ==

### Automatic installation

1. Login to your WordPress Admin Panel. Search for Smart Shipment Tracking through `Plugins > Add New` interface.
2. Find the plugin box of Smart Shipment Tracking and click on the `Install Now` button.
3. Then activate the Smart Shipment Tracking plugin.
4. You will see Smart Shipment Tracking on Left Menu, Click on it. You can configure further. See FAQ's for more information
5. Still have any question? Write us at support@triadmark.com

### Manual installation

1. Download [Smart Shipment Tracking](https://downloads.wordpress.org/plugin/smart-shipment-tracking.zip)
2. Upload the Smart Shipment Tracking through `Plugins > Add New > Upload` interface or upload branded-sms-pakistan folder to the `/wp-content/plugins/` directory.
3. Activate the Smart Shipment Tracking plugin through the `Plugins` menu in WordPress.
4. You will see Smart Shipment Tracking on Left Menu, Click on it. You can configure further. See FAQ's for more information
5. All Done! Still have any question? Write us at support@triadmark.com

== Frequently Asked Questions ==

= Where will my customer see the tracking info? =

The tracking info and a tracking link to track the order on the shipping provider website will be added to the Shipped (Completed) order status emails. Smart Shipment Tracking will also display the tracking info in my-account area for each order in the order history tab.

= Can I add a shipping provider that is not on your list? =

Yes, you can add custom providers, choose your default shipment provider, Change the providers order in the list and enable only providers that are relevant to you.
